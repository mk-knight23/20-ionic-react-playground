# Design System: 20-ionic-react-playground

## Visual Philosophy
The design follows a **Cyber-Mobile / High-Performance Runtime** aesthetic. It uses a deep "Void" background (Slate-950) with "Indigo-Cyan" (#6366f1 to #22d3ee) gradients, focusing on glassmorphism, glowing edges, and technical precision.

## Design Patterns
-   **Hybrid Cards**: Semi-transparent, blur-heavy containers with internal 1 border-white/5 boundaries to simulate premium mobile UI overlays.
-   **Gradient Kinetic text**: High-contrast uppercase headings with horizontal gradient fills for "Action" keywords.
-   **SDK Icons**: 16x16 modular icon containers with internal 20% indigo glows for technical item scannability.
-   **Void Backgrounds**: Deep Slate-950 foundation with subtle indigo-cyan radial gradients to create atmospheric depth.

## Color Palette
-   **Void Core**: Slate-950 (#020617)
-   **Runtime Indigo**: Primary Accent (#6366f1)
-   **Runtime Cyan**: Secondary Accent (#22d3ee)
-   **Status Emerald**: Stability Indicator (#10b981)

## Interaction Design
-   **OS Entry**: Vertical 30px motion reveals for primary sections to simulate a mobile app launch sequence.
-   **Hover Kinetics**: Scale-up effects on cards with dynamic border-color shifting from slate to indigo.
-   **Tactile 2.0**: 105% scaling for primary CTA buttons with shadow-offset expansion for prominent visual feedback.
