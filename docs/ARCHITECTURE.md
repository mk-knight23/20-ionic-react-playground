# Architecture: 20-ionic-react-playground

## Overview
A high-fidelity cross-platform SDK showcase built with React 19 and Vite 6. Designed for the "HybridOS" ecosystem, it demonstrates universal component compilation across iOS, Android, Web, and Desktop targets using a unified TypeScript codebase.

## Tech Stack
-   **Framework**: React 19
-   **Build Tool**: Vite 6
-   **Styling**: Tailwind CSS v4
-   **Animations**: Framer Motion 12
-   **Icons**: Lucide React

## Core Logic
-   **Runtime Showcase**: Interactive dashboard demonstrating the "Build Once, Deploy Everywhere" philosophy.
-   **Capability Grid**: Modular diagnostic layer for testing native performance, component libraries, and hot reload metrics.
-   **Code Preview Engine**: High-fidelity syntax renderer for demonstrating SDK usage (HybridView, Text, Button).
-   **Platform Orchestrator**: Dynamic layout switches for iOS/Android/Web/Desktop simulation.

## Performance
-   Native React 19 rendering for sub-millisecond interaction latency across mobile viewports.
-   Tailwind v4 native engine for optimized cross-platform CSS extraction.
-   GPU-accelerated backdrop filters and gradient interpolations for high-end mobile UI simulation.
